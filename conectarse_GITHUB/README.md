Contiene un pequeño menu:

    $ echo "**************************MENU PARA SCRIPT GITHUB******************************"
    $ echo "0)Proceso completo.************************************************************"
    $ echo "1)Descargar archivos repositorios.*********************************************"
    $ echo "10)Salir.++++++++++************************************************************"
    $ echo "*******************************************************************************"
